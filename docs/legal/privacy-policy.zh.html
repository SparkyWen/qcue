QCue 隐私政策

最后更新：2026 年 6 月 18 日

QCue（“QCue”“我们”）是一款以“快速捕捉”为核心的个人知识应用：你用文字或语音随手记录想法，再由 你自己选择的大语言模型（LLM）将其提炼成一份私人的 markdown 知识库。本政策说明我们收集哪些信息、 为何收集，以及你拥有的选择。我们在设计上力求让你的数据最小化、仅属于你、并由你掌控。

一句话概括：我们只收集应用运行所必需的信息。我们不使用 任何第三方广告或分析 SDK，不跨应用或网站追踪你，也不出售你的数据。

1. 我们收集的信息

  • 账户信息 —— 你注册使用的电子邮箱，以及经安全哈希处理的密码（或一次性登录链接）， 用于创建并保护你的账户。
  • 你创建的内容 —— 你记录的笔记、想法、语音录音及其转写文本、分享的文字/链接，以及 QCue 由此生成的知识库页面。这是你的“第二大脑”，保存在你的设备上；当你登录时，会同步到我们服务器上 专属于你、彼此隔离的空间。
  • 你添加的服务商 API 密钥（BYOK，自带密钥） —— 如果你选择使用某个 LLM 或云语音 服务商，你输入的 API 密钥会被加密存储。我们不会再次完整显示它，也绝不会以明文写入日志。
  • 运营数据 —— 运行服务所需的基础（非广告类）信息，例如记录你 LLM 调用的用量/费用 台账，以及可能包含 IP 地址与时间戳的标准服务器日志（用于安全与防滥用）。

2. 我们如何使用这些信息

  • 提供核心功能：捕捉、同步、生成知识库、回忆/检索，以及夜间整合（Auto-Dream）。
  • 对你进行身份验证并保障账户安全。
  • 运营、维护、排障，并防范对服务的滥用。

我们不会用你的内容来训练我们自己的模型，也不会将其用于广告。

3. 语音与语音识别

当你口述笔记时，只要操作系统支持，转写将在你的设备本地完成。若本地识别不可用，或你 主动选择更高精度的云端转写，音频可能会被发送至云语音转文字服务商以生成转写文本。应用会在录音前作出 提示。仅转写得到的文本会作为你的笔记保存；原始音频在转写完成后不会被保留。

4. 位置信息（可选，默认关闭）

QCue 可以选择性地为你的捕捉记录附上精确位置。该功能默认关闭，仅在你于"设置"中明确 开启"为捕捉记录附加位置"后才会生效。开启后，应用会在你提交捕捉的瞬间读取设备 GPS 坐标，并将经纬度 与该条记录一并保存，以便日后回溯想法产生的场景。位置信息不会在后台采集，也不会在任何 非你主动提交捕捉的时刻被读取。坐标存储在你的私人 vault 中，仅用于你自己的上下文参考——不用于广告、 不与第三方共享（除通常存储和处理你内容的基础设施及 BYOK 服务商外），也仅在你开启云同步时才会传输 至我们的服务器。

5. 第三方 LLM 与语音服务商（BYOK）

QCue 的智能能力由你自己选择并用自己的 API 密钥连接的服务商提供（例如 OpenAI、Anthropic、 DeepSeek 或任何兼容服务商）。为生成或回忆内容，你笔记中的相关文本会用你的密钥发送给你所配置的服务商， 以便其返回结果。该内容由第三方服务商依据其条款与隐私政策进行处理。我们仅作为代你使用 密钥的通道；请你查阅所选服务商的隐私做法。

6. 存储、安全与隔离

  • 所有网络传输均采用 HTTPS/TLS 加密。
  • API 密钥采用认证加密（AES-GCM）静态存储；在设备上则使用操作系统的安全区/钥匙串，读取密钥前可要求 生物识别解锁。
  • 在我们的服务器上，每个账户的数据在数据库层面相互隔离（行级安全），任一账户都无法访问他人数据。
  • 捕捉到的想法会先存储在本地，即使离线也不会丢失。

没有任何传输或存储方式能做到 100% 安全，但我们采用了行业标准的防护措施。

7. 数据留存与删除

你的内容会一直保留，直到你删除它或注销账户为止。你可以在应用内删除单条内容，也可以在应用内 注销账户；注销账户会从我们的在用系统中移除你已同步的内容与服务商密钥。备份与安全日志 将按滚动周期清除。

8. 我们如何共享信息

我们不出售你的个人信息。我们仅在以下情形共享数据：

  • 与你选择连接的 LLM/语音服务商共享（见第 5 节）；
  • 与代表我们托管和运营服务的基础设施服务商共享，且其负有保密义务；
  • 在法律要求时，或为保护用户与服务的权利、安全而必要时。

9. 你的权利

视你所在地区而定，你可能有权访问、更正、导出或删除你的个人数据，并反对或限制某些处理。若你位于 欧洲经济区或英国（GDPR）或加利福尼亚（CCPA/CPRA），这些权利适用于你；我们不会为跨场景行为广告而 “出售”或“共享”个人信息。如需行使任何权利，请通过下方联系方式与我们联系。

10. 儿童

QCue 并非面向 13 岁以下（或你所在司法辖区规定的相应最低年龄）的儿童，我们也不会在知情情况下收集 其个人信息。

11. 跨境传输

QCue 所依赖的基础设施可能位于与你不同的国家/地区，你所连接的服务商也可能在其他国家/地区处理数据。 在法律要求时，我们会依赖适当的合法传输机制。

12. 本政策的变更

我们可能会不时更新本政策。重大变更将通过更新上方“最后更新”日期来体现，并在适当情况下在应用内通知。

13. 联系我们

有关本政策或你的数据的问题或请求：privacy@qcue.cn。
